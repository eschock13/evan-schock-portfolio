ECR Workflow Redesign

Engineering Change Request Improvement Through SQL

At Supreme Machined Products, I spearheaded a project to modernize the Engineering Change Request (ECR) workflow through SQL automation. The previous system relied on manual data entry, leading to inefficiencies and inconsistencies. I improved the existing SQL queries and optimized the workflow order, automating the tracking of changes, approvals, and documentation. This enhancement improved data integrity, reduced processing time, and increased traceability, leading to a more efficient workflow. My contributions helped the company maintain compliance with industry standards while optimizing internal operations.

Technologies Used

SQL: For improved queries and database connection

PLEX: Custom software with SQL integration

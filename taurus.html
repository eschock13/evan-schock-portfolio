TAURUS

Test Automation and Universal Requirement Unification System

TAURUS is an AI-powered test case automation system designed to streamline automotive testing by generating structured test cases directly from system requirements. The system leverages natural language processing to interpret requirements and produce accurate, executable test cases, reducing manual effort and increasing efficiency in the validation process. TAURUS enhances consistency, minimizes human error, and ensures compliance with industry standards.

Technologies Used

META's Llama 3.1: Open source LLM of choice

Python Libraries

Pandas: For data manipulation and formatting

PyQT: Application UI design and development

Unsloth: For finetuning LLMs on consumer hardware

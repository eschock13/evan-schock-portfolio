Shared Mixed Reality

Senior Capstone Project

For our senior design project, my team and I developed a HoloLens application that enabled real-time shared hologram experiences. The project aimed to improve collaboration in mixed reality by allowing multiple users to view, manipulate, and interact with holograms in a synchronized environment. Built using Unity and C#, the application leveraged networking protocols to ensure seamless data transmission between devices. This project demonstrated the potential of augmented reality in fields like engineering, training, and remote assistance, showcasing how immersive technology can enhance teamwork.

Technologies Used

C# Scripts: For application functionality

Unity: For application design

Visual Studio: To build and push software to the device

Mixed Reality Toolkit: For hololens specific interactions

Photon PUN: For hololens networking and sharing services
